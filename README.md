## 1. O Projeto

O projeto tem como objetivo extrair links e seus respectivos nomes de arquivos Markdown



## 2. Como Instalar

$ npm install -g milaferreira/md-links

## 3. Utilizando
No terminal, insira o seguinte comando:

$ md-links lib/__tests__/Teste.md
